Print concept set

Outputs a concept set into human-readable markdown

Usage

printConceptSet(conceptSet, latexTableFontSize = 8)

Arguments

conceptSet

JSON concept set definition to be printed

latexTableFontSize

Font size to use if output will be converted to PDF via latex
