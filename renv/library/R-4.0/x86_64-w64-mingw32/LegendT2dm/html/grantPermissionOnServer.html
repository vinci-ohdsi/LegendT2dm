grantPermissionOnServer {LegendT2dm}R Documentation

Grant read-permission on server

Description

Grant read-permission on server

Usage

grantPermissionOnServer(
  connectionDetails,
  schema,
  user = "legendt2dm_readonly"
)

Arguments

connectionDetails

An object of type connectionDetails as created using the createConnectionDetails function in the DatabaseConnector package. Can be left NULL if connection is provided.

schema

Schema name for read-permission

user

User name for read-permission

Details

Only PostgreSQL servers are supported.
